### **HOW IT WORKS (SIMPLE)**
1. **ESP32** creates `ROVER_CTRL` hotspot  
2. Phone connects → opens `192.168.4.1`  
3. **HTML5 tank dashboard** loads from ESP32 flash  
4. Touch → **WebSocket** → ESP32 drives **motors + camera**  
5. **Zero lag. 100% local.**
