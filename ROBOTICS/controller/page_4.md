## **PAGE 4: INSTALL REQUIRED LIBRARIES**

### **OPEN LIBRARY MANAGER**
1. **Sketch → Include Library → Manage Libraries…**  
   *(or `Ctrl + Shift + I`)*

---

### **INSTALL 6 LIBRARIES (IN ORDER)**

| # | Library | Search Term | Author | Purpose |
|---|--------|-------------|--------|--------|
| 1 | **ESP Async WebServer** | `ESP Async WebServer` | *me-no-dev* | Fast async web |
| 2 | **AsyncTCP** | `AsyncTCP` | *me-no-dev* | WebSocket base |
| 3 | **esp32-cam** | `esp32-cam` | *Espressif* | MJPEG video stream |
| 4 | **LittleFS_esp32** | `LittleFS` | *lorol* | Store `index.html` |
| 5 | **ElegantOTA** | `ElegantOTA` | *Ayush Sharma* | OTA firmware update |
| 6 | **ArduinoJson** | `ArduinoJson` | *Benoit Blanchon* | Parse joystick JSON |

---

### **INSTALL STEPS (REPEAT FOR EACH)**
1. Type **search term**  
2. Click → **“Install”** (latest version)  
3. Wait → **“Installed”** badge  
4. **Close** when all 6 are green

> **Install in order** → avoids errors.

---

### **VERIFY**
1. **Sketch → Include Library**  
2. All 6 listed → **No red warnings**
