### **TROUBLESHOOTING (COMMON FIXES)**

| Problem | Solution |
|--------|----------|
| **Can’t find `DRONE_CTRL` WiFi** | Re-upload code → reboot ESP32 |
| **Page won’t load at `192.168.4.1`** | Type exactly → clear browser cache |
| **Joysticks lag / no response** | Refresh page → ensure **landscape** |
| **Upload fails** | Hold **BOOT** button during flash |
| **SPIFFS error** | Reinstall **ESP32 Sketch Data Upload** tool |
